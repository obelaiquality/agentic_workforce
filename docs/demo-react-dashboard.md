# Guided Demo Repo: React Dashboard Lite

If you want a predictable demo project for screenshots, walkthroughs, or first-run practice, use the fixture at
[`benchmarks/projects/react-dashboard-lite`](../benchmarks/projects/react-dashboard-lite).

## Why This Repo

- small surface area
- front-end oriented
- easy to explain in a demo
- already lives in this repository

## Suggested Demo Flow

1. Start the desktop app or local API.
2. Open `Projects`.
3. Connect the fixture repo as a local project.
4. Switch to `Work`.
5. Run a bounded prompt such as:

```text
Add a compact status summary card to the dashboard and update tests or docs if needed.
```

6. Review the route.
7. Run the task.
8. Show the result in `Codebase` and `Console`.

## What This Demo Proves

- local repo connection
- route review before execution
- managed worktree execution
- verification-aware workflow
- evidence surfaced in the main product UI
