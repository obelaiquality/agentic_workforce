# Support

## Best Channels

- Usage questions and setup issues: GitHub Issues
- Feature requests: GitHub Issues
- Security reports: follow [`SECURITY.md`](SECURITY.md)

## What To Include

- operating system
- install path used: source checkout, packaged app, or browser preview
- exact command run
- local runtime details
- relevant logs or screenshots

## Project Maturity

This project is still in beta. Some flows, especially experimental channels and autonomy features, may change without long-term compatibility guarantees.
