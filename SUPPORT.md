# Support

## Best Channels

- Usage questions and setup issues: GitHub Issues
- Feature requests: GitHub Issues
- Security reports: follow [`SECURITY.md`](SECURITY.md)

## What To Include

- operating system
- install path used: source checkout, packaged app, or browser preview
- exact command run
- local runtime details
- relevant logs or screenshots

## Support Scope

The desktop app, source install path, and GitHub Release artifacts are the primary supported surfaces. Browser preview, local-runtime, benchmarks, Labs, distillation, and channels are supported through the documented runbooks and support matrix rather than the default first-run path.
