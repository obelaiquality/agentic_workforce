# Roadmap

## Now

- Keep the desktop-first first-run path clear and easy to validate
- Improve stable desktop E2E confidence and demo-media freshness
- Make install, configuration, and troubleshooting docs easier for first-time users

## Next

- Reduce setup friction for local-runtime users
- Harden experimental channels and autonomy surfaces behind clearer trust boundaries
- Improve packaged release confidence across platforms

## Later

- Broader contributor ergonomics for advanced runtime and CI coverage
- Deeper automation around demo capture, release proof, and nightly scenario reporting

This roadmap is directional, not a compatibility promise.
