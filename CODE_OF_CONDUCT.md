# Code Of Conduct

## Our Standard

Participate in a way that is respectful, direct, and constructive. Harassment, intimidation, discrimination, and bad-faith disruption are not acceptable.

## Expected Behavior

- Critique ideas, not people.
- Assume shared intent to improve the product.
- Be explicit about risks, regressions, and uncertainty.
- Respect maintainer time by keeping reports reproducible.

## Unacceptable Behavior

- Personal attacks or derogatory language
- Harassment or stalking
- Publishing private information without consent
- Repeated bad-faith issue or PR spam

## Reporting

Report conduct issues through the contact path in [`SECURITY.md`](SECURITY.md) or [`SUPPORT.md`](SUPPORT.md).

Maintainers may remove, edit, or reject contributions that violate this code of conduct.
